## 中文版

**Kern** 是一款轻量级个人目标管理工具，使用 PHP + MySQL 构建，支持多设备同步、番茄专注、RESTful API 和独立手表视图。  
本项目由 **Xiaobocm**（一个初中生）利用**半天时间**完成，代码简洁，开箱即用。

联系作者：xiaobocm@88.com

### 功能特性

- ✅ **目标管理** – 增删改查、完成状态切换、标签分类
- 🍅 **番茄专注** – 自定义专注时长，关联目标，自动统计
- 📱 **多设备同步** – 响应式适配手机、平板、桌面，设备状态实时监控
- 🔔 **新设备通知** – 首次访问时自动识别并提醒
- 🌐 **RESTful API** – 令牌认证，支持筛选与分页，方便第三方集成
- 🎨 **多主题** – 亮色 / 暗色 / 跟随系统
- 🌍 **多语言** – 内置中文（简体/繁体）、English、日本語、한국어，可扩展
- ⌚ **独立手表视图** – 纯黑背景，仅展示今日目标，适合智能手表或极简模式
