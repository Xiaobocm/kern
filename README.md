## 中文

### 简介
**Kern** 是一个用 PHP + MySQL 写成的轻量级个人目标管理工具。你可以记录每日待办（Flag）、追踪大目标进度、切换主题与背景，并支持多语言（中/英/日/韩等）。  
本项目由 **Xiaobocm** 开发，一个初中生利用半天时间完成的作品。

### 功能特点
- 📝 每日 Flag 管理（增删改查、标记完成）
- 📊 统计仪表盘（日/月/年视图，完成率图表）
- 🎯 大目标追踪（进度条）
- 🖼️ 自定义背景（上传/切换/删除）
- 🌙 明暗主题 + 跟随系统
- 🌐 多语言支持（内置 5 种语言，可扩展）
- 💾 数据库备份（需 `mysqldump`）
- 📱 自适应移动端

1. **准备环境**  
   - PHP 7.4+  
   - MySQL 5.7+  
   - Web 服务器（如 Apache / Nginx）

   将所有文件放置于网站根目录。
